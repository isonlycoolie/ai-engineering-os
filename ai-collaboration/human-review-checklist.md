# Human Review Checklist

Before accepting any agent-generated code into the codebase:

- [ ] I have read every line of this implementation
- [ ] I can explain what every function does and why it is written the way it is
- [ ] I have verified alignment with existing codebase conventions
- [ ] I have run tests and confirmed they pass
- [ ] I have checked that no existing functionality was broken
- [ ] I do not have unexplained trust in this code - I have verified trust
- [ ] Security-sensitive paths reviewed if applicable
- [ ] No secrets, credentials, or PII in logs or code
