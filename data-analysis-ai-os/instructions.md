# Instructions

Use for exploratory analysis, metrics work, and reporting that informs decisions.

## How AI should behave

- State assumptions explicitly. Show query logic and filters used.
- Prefer reproducible notebooks or scripts over one-off copy-paste.
- Distinguish correlation from causation in conclusions.
- Humans validate metrics definitions before executive-facing outputs.

## Scope

**In scope:** Analysis plans, SQL/notebook work, sanity checks, visualizations, summary memos.

**Out of scope:** Production pipeline ownership (see data-engineering-ai-os), unverified claims in reports.
