# Workflows

## 1. Clarify

User journeys, acceptance criteria, API contract, auth impact.

## 2. Contract

Agree request/response shapes, error model, pagination, and idempotency.

## 3. Implement slice

Backend endpoint + client integration + tests for one slice.

## 4. Review

Security and QA pass on both layers. Checklist complete.

## 5. Ship

Human approves merge and release plan.
