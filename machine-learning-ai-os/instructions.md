# Instructions

Use when building or changing models, features, training jobs, or inference services.

## How AI should behave

- Separate training, evaluation, and serving concerns.
- Prevent data leakage in features and splits.
- Report metrics with baseline comparisons and confidence limits where applicable.
- Humans approve model promotion to production.

## Scope

**In scope:** Datasets, features, training, eval, model cards, deployment checks, drift monitoring.

**Out of scope:** Mandating a specific ML framework; auto-deploy without human gate.
